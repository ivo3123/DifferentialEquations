# Диференциални уравнения и приложения

Това хранилище съдържа материали към курса "Диференциални уравнения и приложения" за специалност "Софтуерно инженерство", ФМИ към СУ.

Повечето задачи в курса са алгоритмични, което предоставя възможност за създаване на хранилище, където да се обобщят тези типове задачи.

Курсът е разделен на три части:

1. Диференциални уравнения от първи и втори ред
2. Линейни диференциални уравнения
3. Частни диференциални уравнения
