# Уравнение от вид на линейна функция

## Общ вид

$$y'(x) = f(ax + by(x) + c)$$

където $a, b\in \mathbb{R} \setminus \lbrace 0\rbrace$, $c \in \mathbb{R}$

## Алгоритъм за решаване

Полагаме

$$z(x) = ax + by(x) + c$$

Диференцираме по $x$

$$z'(x) = a + by'(x)$$

$$y' = \dfrac{z'(x) - a}{b}$$

Заместваме в даденото уравнение

$$\dfrac{z'(x) - a}{b} = f(z)$$

$$z' = bf(z) + a$$

Получаваме уравнение с разделени променливи относно $x$ и $z(x)$.

## Конкретен пример

$$y' = (x + y + 1)^2$$

Полагаме

$$z(x) = x + y + 1$$

$$z'(x) = 1 + y'$$

$$y' = z' - 1$$

Заместваме в даденото уравнение

$$z' - 1 = z^2$$

$$z'= z^2 + 1$$

Делим на $z^2 + 1 > 0$

$$\dfrac{z'}{z^2 + 1} = 1$$

Получаваме уравнение с разделящи се променливи относно $x$ и $z(x)$.

Интегрираме по $x$

$$\int \dfrac{1}{z^2 + 1} \space dz = \int dx$$

$$\arctan{z} = x + C$$

$$z(x) = \tan(x + C) \qquad x + C \in \left(-\dfrac{\pi}{2}, \dfrac{\pi}{2}\right)$$

$$y(x) = \tan(x + C) - x - 1 \qquad x + C \in \left(-\dfrac{\pi}{2}, \dfrac{\pi}{2}\right)$$
