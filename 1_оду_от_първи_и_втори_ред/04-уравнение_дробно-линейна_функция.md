# Уравнение от вид на дробно-линейна функция

## Общ вид

$$y'(x) = f\left(\dfrac{ax + by(x) + c}{mx + ny(x) + p}\right)$$

където $a, b, c, m, n, p \in \mathbb{R}$

## Алгоритъм за решаване

Разглежадаме изразите в числител и в знаменател и ги приравняваме на $0$, за да ги интерпретираме като аналитични прави в равнината.

$$
\begin{cases}
l_1 := ax + by(x) + c = 0 \newline
l_2 := mx + ny(x) + p = 0
\end{cases}
$$

### Съвпадащи прави

Ако $l_1 \equiv l_2$, то можем да напишем $a = km$, $b = kn$ и $c = kp$, където $k$ е реално число, което удовлетворява условието за съвпадане на правите.

$$y'(x) = f\left(\dfrac{kmx + kny(x) + kp}{mx + ny(x) + p}\right)$$

$$y'(x) = f\left(k\dfrac{mx + ny(x) + p}{mx + ny(x) + p}\right)$$

$$y'(x) = f(k)$$

Уравнението става тривиално и частен случай на уравнение с разделени променливи.

### Успоредни прави

Ако $l_1 || l_2$, то можем да напишем $a = km$ и $b = kn$, където $k$ е реално число, което удовлетворява условието за успоредност на правите.

$$y'(x) = f\left(\dfrac{kmx + kny(x) + c}{mx + ny(x) + p}\right)$$

$$y'(x) = f\left(k\dfrac{mx + ny(x) + \frac{c}{k}}{mx + ny(x) + p}\right)$$

Представяме

$$\frac{c}{k} = p + q$$

$$y'(x) = f\left(k\dfrac{mx + ny(x) + p + q}{mx + ny(x) + p}\right)$$

$$y'(x) = f\left(k\dfrac{mx + ny(x) + p}{mx + ny(x) + p} + k\dfrac{q}{mx + ny(x) + p}\right)$$

$$y'(x) = f\left(k + k\dfrac{q}{mx + ny(x) + p}\right)$$

Получаваме уравнение от вид на линейна функция от вида

$$y'(x) = \phi(mx + ny + p)$$

### Пресичащи се прави

Ако системата е определена, то я решаваме и намираме решението и $(x_0, y_0)$.

Целта ни е да транслираме координатната система $(x, y)$ с център точката $(0, 0)$ до нова координатна система $(u, v)$ с център точката $(x_0, y_0)$.

Нека

$$u = x - x_0$$

$$v = y - y_0$$

Изразяваме

$$x = u + x_0$$

$$y(x) = v(u) + y_0$$

$$y'(x) = v'(u)$$

Заместваме в даденото уравнение

$$v' = f\left(\dfrac{a(u+x_0) + b(v+y_0) + c}{m(u+x_0) + n(v+y_0) + p}\right)$$

$$v' = f\left(\dfrac{au + bv + [ax_0 + by_0 + c]}{mu + nv + [mx_0 + ny_0 + p]}\right)$$

Изразите в скобите са точно решенията на системата и съответно са равни на $0$.

$$v' = f\left(\dfrac{au + bv}{mu + nv}\right)$$

Делим в числител и в знаменател на $u \ne 0$.

$$v' = f\left(\dfrac{a + b\frac{v}{u}}{m + n\frac{v}{u}}\right)$$

Получаваме хомогенно уравнение относно $u$ и $v(u)$.

Полагаме

$$z(u) = \dfrac{v(u)}{u}$$

$$v = zu$$

$$v' = z'u + z$$

Заместваме в уравнението

$$z'u + z = f\left(\dfrac{a + bz}{m + nz}\right)$$

$$z' = \dfrac{1}{u}\left[f\left(\dfrac{a + bz}{m + nz}\right) - z\right]$$

Получаваме уравнение с разделени променливи относно $u$ и $z(u)$

## Конкретен пример

$$y' = \dfrac{3y-8x+2}{y-3x+1}$$

$$
\begin{cases}
3y-8x+2 = 0 \newline
y-3x+1 = 0
\end{cases}
$$

$$(x_0, y_0) = (1, 2)$$

Нека

$$u = x - 1$$

$$v = y - 2$$

Изразяваме:

$$x = u + 1$$

$$y = v + 2$$

$$y'(x) = v'(u)$$

Заместваме в даденото уравнение

$$v' = \dfrac{3v - 8u}{v - 3u}$$

Делим на $u$ в числител и в знаменател

$$v' = \dfrac{3\frac{v}{u} - 8}{\frac{v}{u} - 3}$$

Получаваме хомогенно уравнение относно $u$ и $v(u)$.

$$z(u) = \dfrac{v(u)}{u}$$

$$v' = z'u + z$$

Заместваме в уравнението

$$z'u + z = \dfrac{3z - 8}{z - 3}$$

$$z'u = \dfrac{3z - 8}{z - 3} - z$$

$$z'u = \dfrac{3z - 8}{z - 3} - \dfrac{z^2-3z}{z-3}$$

$$z'u = \dfrac{-z^2 + 6z - 8}{z - 3}$$

$$z' = -\dfrac{1}{u}\dfrac{z^2 - 6z + 8}{z - 3}$$

$$z' = -\dfrac{1}{u}\dfrac{(z-2)(z-4)}{z - 3}$$

Получаваме уравнение с разделени променливи относно $u$ и $z(u)$

Искаме да разделим на $(z-2)(z-4)$. Ще проверим кога двата множителя се зануляват.

Нека $z-2 = 0$

$$z = 2$$

$$\dfrac{v}{u} = 2$$

$$\dfrac{y-2}{x-1} = 2$$

$$y-2 = 2x-2$$

$$y = 2x$$

Аналогично нека $z - 4 = 0$

$$y = 4x - 2$$

Така $y = 2x$ и $y = 4x - 2$ са решения на диференциалното уравнение.

Делим на $z \ne 2$ и $z \ne 4$

$$\dfrac{z - 3}{(z-2)(z-4)}z' = -\dfrac{1}{u}$$

Интегрираме по $u$

$$\int \dfrac{z - 3}{(z-2)(z-4)}z' \space du = \int \left(-\dfrac{1}{u}\right) \space du$$

$$\int \dfrac{z - 3}{(z-2)(z-4)} \space dz = - \int \dfrac{1}{u} \space du$$

Ще решим левия интеграл като изпозваме, че $z - 3$ е "център" за $z-2$ и $z-4$

$$\int \dfrac{z - 3}{(z-3+1)(z-3-1)} \space d(z-3) = - \int \dfrac{1}{u} \space du$$

$$\int \dfrac{z - 3}{(z-3)^2-1} \space d(z-3) = - \int \dfrac{1}{u} \space du$$

$$\dfrac{1}{2} \int \dfrac{1}{(z-3)^2-1} \space d[(z-3)^2 - 1] = - \int \dfrac{1}{u} \space du$$

$$\dfrac{1}{2} \ln{|(z-3)^2-1|} = -\ln{|u|} + C$$

$$\ln{\sqrt{|(z-3)^2-1|}} = \ln{\left|\frac{1}{u}\right|} + \ln{|C_1|}$$

$$\ln{\sqrt{|(z-3)^2-1|}} = \ln{\left|\frac{C_1}{u}\right|}$$

$$\sqrt{|(z-3)^2-1|} = \left|\frac{C_1}{u}\right|$$

$$|(z-3)^2-1| = \left|\frac{C_1}{u}\right|^2$$

$$|(z-3)^2-1| = \left(\frac{C_1}{u}\right)^2$$

$$(z-3)^2-1 = \pm \dfrac{C_1^2}{u^2}$$

$$(z-3)^2-1 = \dfrac{C_2}{u^2}$$

$$\left(\dfrac{v}{u}-3\right)^2-1 = \dfrac{C_2}{u^2}$$

$$\left(\dfrac{y-2}{x-1}-3\right)^2-1 = \dfrac{C_2}{(x-1)^2}$$

$$\left(\dfrac{y-2}{x-1}\right)^2 - 6\dfrac{y-2}{x-1} + 9 - 1 = \dfrac{C_2}{(x-1)^2}$$

$$(y-2)^2 - 6(x-1)(y-2) + 8(x-1)^2 - C_2 = 0$$

$$y_{1,2}-2 = 3(x-1) \pm \sqrt{9(x-1)^2 - (8(x-1)^2 - C_2)}$$

$$y_{1,2} = 3x - 1 \pm \sqrt{(x-1)^2 + C_2}$$
