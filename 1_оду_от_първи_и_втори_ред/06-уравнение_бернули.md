# Уравнение на Бернули

## Общ вид

$$y'(x) = a(x)y(x) + b(x)y^n$$

където $a, b: \mathbb{R} \to \mathbb{R}$ и $n \in \mathbb{Z}$

## Алгоритъм за решаване

Ако $n = 0$ получаваме линейно уравнение.

Ако $n = 1$ получаваме уравнение с разделящи се променливи.

Вижда се, че $y \equiv 0$ е решение при $n > 0$, иначе не е допустима стойност.

Делим на $y^n \ne 0$.

$$\dfrac{y'}{y^n} = a(x)y^{1-n}+b(x)$$

Полагаме

$$z = y^{1-n}$$

Диференцираме по $x$

$$z' = (1-n)y^{-n}y'$$

$$y' = \dfrac{z'y^n}{1-n}$$

Заместваме в даденото уравнение.

$$\dfrac{z'y^n}{(1-n)y^n} = a(x)z+b(x)$$

$$z' = (1-n)a(x)z+(1-n)b(x)$$

Получаваме линейно уравнение относно $x$ и $z(x)$.

## Конкретен пример

$$y' = \dfrac{1}{2x}y -\dfrac{x}{2}y^{-1}$$

$y \ne 0$ по условие

Делим на $y^{-1}$.

$$y'y = \dfrac{1}{2x}y^2 - \dfrac{x}{2}$$

Полагаме

$$z(x) = y^2$$

$$z' = 2yy'$$

$$y' = \dfrac{z'}{2y}$$

Заместваме в уравнение.

$$\dfrac{z'}{2y}y = \dfrac{1}{2x}z - \dfrac{x}{2}$$

$$z' = \dfrac{1}{x}z - x$$

Получаваме линейно уравнение относно $x$ и $z(x)$.

Решението има вида

$$z(x) = \displaystyle e^{I}\left( C + \int (-x) e^{-I} \space dx \right)$$

където

$$I = \displaystyle \int \dfrac{1}{x} \space dx = \ln{|x|}$$

$$-I = -\ln{|x|} = \ln{\left|\frac{1}{x}\right|}$$

Така

$$z(x) = \displaystyle e^{\ln{|x|}}\left( C + \int (-x) e^{\ln{\left|\frac{1}{x}\right|}} \space dx \right)$$

$$z(x) = \displaystyle |x|\left( C + \int (-x) \left|\frac{1}{x}\right| \space dx \right)$$

Знаем, че $\text{sign}(x) = \text{sign}\left(\frac{1}{x}\right)$. Това условие гарантира, че за всяко $x$ разкриването на модулите ни дава един и същи знак

$$z(x) = \displaystyle \pm x \left( C + \int (-x) \left(\pm\frac{1}{x}\right) \space dx \right)$$

Умножението на $\pm$ с $\pm$ ни дава $+$, тъй като са еднакви знаци.

$$z(x) = \displaystyle x \left(\pm C + \int (-x) \left(\frac{1}{x}\right) \space dx \right)$$

$$z(x) = \displaystyle x \left(C_1 - \int dx \right)$$

$$z(x) = x(C_1 - x)$$

$$y^2(x) = C_1x - x^2$$

$$y(x) = \pm \sqrt{C_1x - x^2}$$
