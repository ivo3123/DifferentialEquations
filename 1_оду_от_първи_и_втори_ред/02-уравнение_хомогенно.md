# Хомогенно уравнение

## Общ вид

$$y'(x) = F\left(\dfrac{y(x)}{x}\right)$$

Уравнението е хомогенно по $x$ и $y$, тоест ако запишем уравнението във вида

$$g(x, y, y') = 0$$

то за $k \in \mathbb{R} \setminus \{0\}$ е изпълнено, че

$$g(x, y, y') = 0 \iff g(kx, ky, y') = 0$$

## Алгоритъм за решаване

Свеждаме уравнението до такъв вид, че да има само $\dfrac{y}{x}$ или $\dfrac{x}{y} = \dfrac{1}{\frac{y}{x}}$.

Полагаме

$$z(x) = \dfrac{y(x)}{x}$$

$$y(x) = z(x) \space x$$

Диференцираме по $x$

$$y' = z'x + zx'$$

$$y' = z'x + z$$

Заместваме в уравнението

$$z'x + z = F(z)$$

$$z' = \dfrac{F(z) - z}{x}$$

Свеждохме до уравнение с разделящи се променливи за $x$ и $z(x)$.

## Конкретен пример

$$2xyy' = y^2 - x^2$$

$y \equiv 0$ не е решение на уравнението

Делим на $2xy$

$$y' = \dfrac{1}{2} \left(\dfrac{y}{x} - \dfrac{x}{y}\right)$$

Нека

$$z(x) = \dfrac{y(x)}{x}$$

$$y = zx$$

$$y' = z'x + z$$

Заместваме в уравнението

$$z'x + z = \dfrac{1}{2} \left(z - \dfrac{1}{z}\right)$$

След еквивалентни преобразувания получаваме следното уравнение с разделени променливи относно $x$ и $z(x)$

$$2z' = -\dfrac{1}{x}\dfrac{z^2+1}{z}$$

Делим на $z^2+1 > 0$ и умножаваме по $z$

$$\dfrac{2zz'}{z^2 + 1} = -\dfrac{1}{x}$$

Интегрираме по $x$.

$$\displaystyle \int \dfrac{2z}{1 + z^2}\mathrm{d}z = - \int \dfrac{1}{x}\mathrm{d}x$$

$$\displaystyle \int \dfrac{\mathrm{d}(z^2)}{1 + z^2} = - \int \dfrac{\mathrm{d}x}{x}$$

$$\ln|1 + z^2| = -\ln|x| + C_0$$

$$e^{\ln|1 + z^2|} = e^{-\ln|x| + C_0}$$

$$e^{\ln|1 + z^2|} = e^{\ln|x|^{-1}}e^{C_0}$$

$$|1 + z^2| = \dfrac{1}{|x|}e^{C_0}$$

$$1 + z^2 = \pm\dfrac{1}{x}e^{C_0}$$

$$C = \pm e^{C_0}$$

$$z^2 = \dfrac{C}{x} - 1$$

$$\dfrac{y^2}{x^2} = \dfrac{C}{x} - 1$$

$$y^2 = Cx - x^2$$

$$y = \pm \sqrt{Cx - x^2}$$
