# Уравнение, нерешено относно производната

## Общ вид

$$y(x) = f(x, y'(x))$$

## Алгоритъм за решаване

Полагаме $z(x) = y'(x)$

Получаваме $y(x) = f(x, z(x))$

Диференцираме по $x$.

Очакваме да получим израз, който позволява факторизация.

__*ВАЖНО*__! В крайния отговор трябва да има точно 1 произволна константа, тъй като това е ОДУ от първи ред!

## Конкретен пример

$$5y = xy' + x^2 - (y')^2$$

Полагаме

$$z(x) = y'(x)$$

Заместваме в уравнението

$$5y = xz + x^2 - z^2 \quad (*)$$

Диференцираме по $x$.

$$5y' = z+xz'+2x-2zz'$$

$$5z = z+xz'+2x-2zz'$$

$$4z + 2zz' = xz'+2x$$

$$2z(2 + z') = x(z'+2)$$

$$2z(z' + 2) - x(z'+2) = 0$$

$$(2z - x)(z' + 2) = 0$$

$$(x-2z)(z'+2) = 0$$

Получаваме произведение да е равно на нула. Това означава, че някой от множителите е нула.

Разглеждаме следните уравнения от по-нисък ред:

$$x-2z = 0$$

$$z'+2 = 0$$

Като намерим $z$ от всяко от двете уравнения, заместваме в $(*)$.

**I:** уравнението $x-2z = 0$

$$z = \dfrac{x}{2}$$

$$5y = x\dfrac{x}{2} + x^2 - \dfrac{x^2}{4}$$

$$5y = \dfrac{2x^2}{4} + \dfrac{4x^2}{4} - \dfrac{x^2}{4}$$

$$5y = \dfrac{5x^2}{4}$$

$$y = \dfrac{1}{4}x^2$$

**II:** уравнението $z'+2 = 0$

$$z' = -2$$

$$z = -2x + C$$

$$5y = x(-2x + C) + x^2 - (-2x + C)^2$$

$$5y = -2x^2 + Cx + x^2 - 4x^2 + 4Cx - C^2$$

$$5y = -5x^2 + 5Cx - C^2$$

$$y = -x^2 + Cx - \dfrac{C^2}{5}$$
