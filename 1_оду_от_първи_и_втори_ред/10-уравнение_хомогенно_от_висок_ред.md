# Хомогенно уравнение от по-висок ред

## Общ вид

$$f(x, y, y', y'') = 0$$

Уравнението е хомогенно по $y$ и производните на $y$. За $k \in \mathbb{R}\setminus \lbrace 0\rbrace$ е изпълнено, че

$$f(x, y, y', y'') = 0 \iff f(x, ky, ky', ky'') = 0$$

## Алгоритъм за решаване

Проверяваме дали $y \equiv 0$ е решение.

Полагаме

$$z(x) = \dfrac{y'(x)}{y(x)}$$

$$y'(x) = z(x)y(x)$$

Диференцираме по $x$

$$y'' = z'y + zy'$$

$$y'' = z'y + z^2y$$

$$y'' = y(z' + z^2)$$

Заместваме в уравнението и получаваме

$$f(x, y, zy, y(z'+z^2)) = 0$$

Знаем, че уравнението е хомогенно по $y$ и производните на $y$, тоест за $y \ne 0$ уравнението се свежда до

$$f(x, 1, z, z'+z^2) = 0$$

Получаваме уравнение от по-нисък ред за $x$ и $z(x)$

## Конкретен пример

$$yy'' = (y')^2 + 6y^2x$$

Вижда се, че $y \equiv 0$ е решение

Полагаме

$$z(x) = \dfrac{y'(x)}{y(x)}$$

$$y' = zy$$

$$y'' = y(z' + z^2)$$

Заместваме в даденото уравнение.

$$yy(z' + z^2) = z^2y^2 + 6y^2x$$

$$y^2(z' + z^2) = z^2y^2 + 6y^2x = 0$$

Делим на $y^2 \ne 0$

$$z' + z^2 = z^2 + 6x$$

$$z' = 6x$$

Получаваме уравнение с разделени променливи за $x$ и $z(x)$

Интегрираме по $x$

$$\displaystyle \int z' \space dx = \int 6x \space dx$$

$$z = 3x^2 + C_1$$

$$\dfrac{y'}{y} = 3x^2 + C_1$$

Получаваме уравнение с разделени променливи за $x$ и $y(x)$

Интегрираме по $x$

$$\displaystyle \int \dfrac{y'}{y} \space dx = \int (3x^2 + C_1) \space dx$$

$$\displaystyle \int \dfrac{1}{y} \space dy = \int 3x^2 \space dx + \int C_1 \space dx$$

$$\ln{|y|} = x^3 + C_1x + C_2$$

$$e^{\ln{|y|}} = e^{x^3 + C_1x + C_2}$$

$$|y| = e^{C_2}e^{x^3 + C_1x}$$

$$y = \pm e^{C_2}e^{x^3 + C_1x}$$

$$y = C_3 e^{x^3 + C_1x}$$
