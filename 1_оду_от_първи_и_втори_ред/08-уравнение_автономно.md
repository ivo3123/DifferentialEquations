# Автономно уравнение

## Общ вид

$$f(y, y', y'') = 0$$

$x$ не участва явно.

## Алгоритъм за решаване

Полагаме

$$y'(x) = P(y(x))$$

Диференцираме по $x$

$$(y'(x))' =  (P(y(x)))'$$

$$y''(x) = P'(y(x))y'(x)$$

$$y'' = P'P$$

Заместваме в уравнението и получаваме уравнение от по-нисък ред.

Интегрира се по $y$. (Това е единственият тип задача, където се интегрира по $y$)

## Конкретен пример

$$(y')^2 + 2yy'' = 0$$

Полагаме

$$y'(x) = P(y(x))$$

$$y''(x) = P'P$$

Заместваме в уравнението

$$P^2 + 2yPP' = 0$$

$$P(P + 2yP') = 0$$

Получаваме произведение да е равно на нула. Това означава, че някой от множителите е нула.

Нека разгледаме първото уравнението

$$P = 0$$

$$y' = 0$$

$$y = C_1$$

Нека разгледаме второто уравнение

$$P + 2yP' = 0$$

$$2yP' = -P$$

Знаем, че $y = C_1$ е решение, в частност и $y = 0$ е решение. Делим на $y \ne 0$

$$P' = -\dfrac{1}{2y}P$$

Получаваме уравнение с разделени променливи относно $y(x)$ и $P(y(x))$

Делим на $P \ne 0$

$$\dfrac{P'}{P} = -\dfrac{1}{2y}$$

Интегрираме по $y$

$$\displaystyle \int \dfrac{P'}{P} \space dy = \int \left(-\dfrac{1}{2y}\right) \space dy$$

$$\displaystyle \int \dfrac{1}{P} \space dP = - \dfrac{1}{2} \int \dfrac{1}{y} \space dy$$

$$\ln{|P|} = -\dfrac{1}{2}\ln{|y|} + C_2$$

$$\ln{|P|} = \ln{\left|\dfrac{1}{\sqrt{y}}\right|} + \ln{|C_3|}$$

$$\ln{|P|} = \ln{\left|\dfrac{C_3}{\sqrt{y}}\right|}$$

$$|P| = \left|\dfrac{C_3}{\sqrt{y}}\right|$$

$$P = \pm \dfrac{C_3}{\sqrt{y}}$$

$$y'\sqrt{y} = C_4$$

Интегрираме по $x$

$$\displaystyle \int y'\sqrt{y} \space dx = \int C_4 \space dx$$

$$\displaystyle \int \sqrt{y} \space dy = C_4 \int dx$$

$$\dfrac{\sqrt{y^3}}{\frac{3}{2}} = C_4x + C_5$$

$$\dfrac{2}{3}\sqrt{y^3} = C_4x + C_5$$

$$\sqrt{y^3} = C_6x + C_7$$

$$y = \sqrt[3]{(C_6x + C_7)^2}$$
