# Уравнение в точна производна

## Общ вид

$$f(x, y, y', y'') = 0$$

и съществува функция $F(x, y, y')$, за която е изпълнено, че

$$F'_x(x, y, y') = f(x, y, y', y'')$$

## Алгоритъм за решаване

За да намерим $F$, интегрираме $f$ по $x$ и очакваме интегралите да се разкрият и да се получи функция в явен вид.

$$\int f(x, y, y', y'') \space dx = C$$

Така

$$F(x, y, y') = C$$

Получаваме уравнение от по-нисък ред.

## Конкретен пример

$$\dfrac{y'}{x} - \dfrac{y}{x^2} = \cos{x} - y''$$

Интегрираме двете страни по $x$

$$\int \left(\dfrac{y'}{x} - \dfrac{y}{x^2}\right) \space dx = \int (\cos{x} - y'') \space dx$$

$$\int \dfrac{y'}{x} \space dx - \int \dfrac{y}{x^2} \space dx = \int \cos{x} \space dx - \int y'' \space dx$$

В първия и в четвъртия интеграл ще внесем съответно $y'$ и $y''$ под диференциала, като ги интегрираме. Третият интеграл има дирекрна примивна.

$$\int \dfrac{1}{x} \space dy - \int \dfrac{y}{x^2} \space dx = \sin{x} - \int dy'$$

Първият интеграл ще го разбием по части. Вторият интеграл за сега го оставяме в такъв вид. Последният интеграл има дирекрна примивна.

$$\dfrac{y}{x} - \int y \space d\frac{1}{x} - \int \dfrac{y}{x^2} \space dx = \sin{x} - y'$$

От левия интеграл ще изнесем $\frac{1}{x}$ от диференциала, като го интегрираме.

$$\dfrac{y}{x} + \int \dfrac{y}{x^2} \space dx - \int \dfrac{y}{x^2} \space dx = \sin{x} - y'$$

$$\dfrac{y}{x} = \sin{x} - y' + C$$

$$y' = -\dfrac{1}{x}y + \sin{x} + C$$

Получаваме линейно уравнение от първи ред относно $x$ и $y(x)$

$$y(x) = \displaystyle e^{I}\left( K + \int (\sin{x} + C) e^{-I} \space dx \right)$$

където

$$I = \displaystyle - \int \dfrac{1}{x} \space dx = -\ln{|x|} = \ln{\left|\frac{1}{x}\right|}$$

$$-I = \ln{|x|}$$

Така

$$y(x) = \displaystyle e^{\ln{\left|\frac{1}{x}\right|}}\left( K + \int (\sin{x} + C) e^{\ln{|x|}} \space dx \right)$$

$$y(x) = \displaystyle \left|\frac{1}{x}\right|\left( K + \int (\sin{x} + C) |x| \space dx \right)$$

Знаем, че $\text{sign}(x) = \text{sign}\left(\frac{1}{x}\right)$. Това условие гарантира, че за всяко $x$ разкриването на модулите ни дава един и същи знак

$$y(x) = \displaystyle \pm \frac{1}{x} \left( K + \int (\sin{x} + C) (\pm x) \space dx \right)$$

Умножението на $\pm$ с $\pm$ ни дава $+$, тъй като са еднакви знаци.

$$y(x) = \displaystyle \frac{1}{x} \left( \pm K + \int (\sin{x} + C) x \space dx \right)$$

$$y(x) = \displaystyle \frac{1}{x} \left( K_1 + \int x\sin{x} \space dx + \int C x \space dx \right)$$

$$y(x) = \displaystyle \frac{1}{x} \left( K_1 - \int x \space d\cos{x} + C \int x \space dx \right)$$

$$y(x) = \displaystyle \frac{1}{x} \left( K_1 - \left(x\cos{x} - \int \cos{x} \space dx\right)+ C \dfrac{x^2}{2} \right)$$

$$y(x) = \displaystyle \frac{1}{x} \left( K_1 - \left(x\cos{x} - \sin{x} \right)+ C \dfrac{x^2}{2} \right)$$

$$y(x) = \displaystyle \frac{1}{x} \left( K_1 - x\cos{x} + \sin{x} + C \dfrac{x^2}{2} \right)$$
